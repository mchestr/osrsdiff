# Product Overview

OSRS Diff is a backend service for tracking Old School RuneScape character progression by periodically fetching hiscores data from the official OSRS API.

## Core Features

- **Player Tracking**: Add/remove OSRS players for continuous monitoring
- **Automated Data Collection**: Background workers fetch hiscore data at configurable intervals
- **Progress Analysis**: Historical progression tracking with experience gains, level changes, and boss kill counts
- **RESTful API**: Secure JWT-authenticated endpoints for player management and statistics
- **Real-time Updates**: Manual fetch triggers for immediate data updates

## Target Users

- OSRS players wanting to track their own progression over time
- Clan leaders monitoring member progress
- Content creators analyzing player statistics
- Developers building OSRS-related applications

## Key Value Propositions

- Automated tracking eliminates manual hiscore checking
- Historical data enables trend analysis and goal tracking
- Secure API allows integration with external applications
- Reliable background processing ensures consistent data collection
